# Zepto SQL Data Analysis

This project focuses on performing end-to-end data analysis using PostgreSQL on a real-world inventory dataset inspired by Zepto (quick commerce domain). The goal of this project is to extract meaningful business insights from raw data using SQL by applying data cleaning, transformation, and advanced query techniques.

It simulates how data analysts work with real datasets in e-commerce or quick delivery platforms—understanding product performance, pricing strategies, and revenue patterns.

---

## Project Overview

In this project, raw inventory data was taken and processed step by step to make it analysis-ready. The dataset includes product-level information such as pricing, discounts, categories, and availability.

The analysis answers key business questions like:

* Which products generate the most revenue?
* How do discounts impact sales?
* Which categories perform best?
* What pricing strategies are being used?

---

## Key Highlights

* Data Cleaning & Transformation

  * Removed inconsistencies and handled missing/null values
  * Standardized column formats for better analysis
  * Prepared structured tables for querying

* Exploratory Data Analysis (EDA)

  * Analyzed product distribution across categories
  * Identified pricing patterns and discount trends
  * Examined inventory and availability insights

* Revenue & Discount Analysis

  * Calculated total revenue generated by products and categories
  * Measured the impact of discounts on sales performance
  * Identified high-performing and underperforming items

* Advanced SQL Queries

  * Used JOINs to combine multiple datasets
  * Applied GROUP BY for aggregation and summaries
  * Implemented WINDOW FUNCTIONS (e.g., RANK, DENSE_RANK) for deeper insights
  * Created complex queries to simulate real-world business scenarios

* Business Insights Generation

  * Identified top-selling products and categories
  * Highlighted inefficient discount strategies
  * Suggested data-driven improvements for pricing and inventory decisions
